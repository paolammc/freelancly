# Freelancly Design Files

This folder contains Pencil.dev design files (`.pen`) for the Freelancly application.

## Setup

1. Install the Pencil VS Code extension
2. Open any `.pen` file to launch the visual canvas
3. Use Cmd+K to access AI prompts for design modifications

## Design Files

- `dashboard.pen` - Freelancer dashboard layout
- `sidebar.pen` - Navigation sidebar component
- `profile.pen` - Freelancer profile page
- `branding.pen` - Brand colors, typography, and tokens

## Workflow

1. Design visually on the Pencil canvas
2. Use AI prompts to generate React + Tailwind code
3. Copy generated code into your components
